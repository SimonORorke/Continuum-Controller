#### **Continuum Controller v2.0**
##### **by Richard Kram and Simon O'Rorke.** 
------
**Supported Instruments:**

- Haken Continuum (Slim models)
- Haken Continuum (older “Thick” models with at least at least 2x DSP)
- EaganMatrix Eurorack module

------

**Supported Haken Audio Firmware Versions:**  10.44 and later.

------

**Electra One Requirements**

-  Hardware: Mk II (this preset will not run on Mk1 models.)
-  Firmware: 4.0 or later

------

**Superseded Electra One Presets**
- Continuum Ctrl 10.52
- EMM Ctrl 10.52.

------

*Do not run with the Haken Editor.* Inconsistent operation may result.

Connect to Slim Continuum using Electra One Port 1 Midi DIN Output/Input to Continuum Midi (using TRS adapters).

**Documentation**

The **User Guide** can be downloaded from the first post of [Continuum Controller's Electra One Forum topic](https://forum.electra.one/t/continuum-controller). It will also eventually be available in the User Guides folder of Haken Editor/Firmware distributions.

This [Video on Operation](https://www.youtube.com/watch?v=MzKwRkp7vH8&t=372s) was made for an earlier version but is still very useful.
